# Template Preview Gallery

Visual previews of all barbershop template variants.

## How to Generate Screenshots

Screenshots can be generated by running each template locally and capturing the homepage.

### Method 1: Manual Screenshots

```bash
# For each variant, run:
cd sites/templates/barbershop/variant-1-modern
npm install
npm run dev

# Visit http://localhost:3000
# Capture screenshot at 1920x1080 resolution
# Save as: previews/barbershop-variant-1.png
```

### Method 2: Using Playwright (Automated)

Create `generate-previews.js` in the project root:

```javascript
const { chromium } = require('playwright');
const path = require('path');

async function captureScreenshot(variant, port, filename) {
  const browser = await chromium.launch();
  const page = await browser.newPage({
    viewport: { width: 1920, height: 1080 }
  });
  
  await page.goto(`http://localhost:${port}`);
  await page.waitForTimeout(2000); // Wait for animations
  
  await page.screenshot({
    path: path.join(__dirname, 'previews', filename),
    fullPage: false
  });
  
  await browser.close();
  console.log(`âœ“ Captured ${filename}`);
}

async function main() {
  // Start each dev server and capture
  // variant-1-modern: port 3000
  // variant-2-luxury: port 3001
  // variant-3-classic: port 3002
  
  await captureScreenshot('modern', 3000, 'barbershop-variant-1.png');
  await captureScreenshot('luxury', 3001, 'barbershop-variant-2.png');
  await captureScreenshot('classic', 3002, 'barbershop-variant-3.png');
}

main();
```

Then run:

```bash
npm install playwright
# Start all dev servers in separate terminals
node generate-previews.js
```

## Template Variants

### Variant 1: Modern

**File:** `barbershop-variant-1.png`

- Clean, minimalist design
- White/light gray background
- Deep blue accents (#1E40AF)
- Sans-serif fonts (Inter, Poppins)
- Single column layout with whitespace

**Best for:** Modern barbershops targeting urban professionals

---

### Variant 2: Luxury

**File:** `barbershop-variant-2.png`

- Premium, sophisticated design
- Dark theme (#1A1A1A background)
- Gold accents (#D4AF37)
- Serif fonts for headers (Playfair Display)
- Full-width hero with elegant animations

**Best for:** High-end barbershops, executive clientele

---

### Variant 3: Classic

**File:** `barbershop-variant-3.png`

- Traditional barbershop aesthetic
- Warm earth tones (browns, creams)
- Red/white barber pole colors
- Vintage card styling
- Grid-based layout

**Best for:** Traditional neighborhood barbershops, family-friendly
