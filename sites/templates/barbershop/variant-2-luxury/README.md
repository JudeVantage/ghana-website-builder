# Barbershop Template - Luxury Variant

Premium, sophisticated design with dark theme and gold accents.

## Design Features
- **Color Scheme**: Dark background (#1A1A1A) with gold (#D4AF37) accents
- **Typography**: Playfair Display serif for headers, Inter for body
- **Layout**: Full-width hero with split sections
- **Style**: Elegant, luxurious, upscale

## Sections
- Full-width hero with animated gold particles
- Services grid (2 columns) with hover effects
- About with staggered image grid
- Gallery (8 images, 4-column grid) with overlays
- Contact information with styled map
- Luxury footer with decorative elements
- Gold WhatsApp floating button
- Premium booking modal

## Installation

```bash
npm install
npm run dev
```

Opens on port 3001 to avoid conflicts.

## Customization

All placeholder content is marked with `{{VARIABLE_NAME}}` for easy replacement via the deployment script.

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons
