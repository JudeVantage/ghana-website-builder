# Barbershop Template - Classic Variant

Traditional barbershop design with warm earth tones and red/white barber pole aesthetic.

## Design Features
- **Color Scheme**: Warm earth tones (browns, creams) with red (#DC2626) accents
- **Typography**: Georgia serif and Impact display fonts
- **Layout**: Grid-based, classic structure with vintage cards
- **Style**: Traditional, nostalgic, authentic barbershop feel

## Sections
- Hero with barber pole stripes and vintage badge
- Services grid (2 columns) with shadow box cards
- About with photo collage
- Gallery (8 images, 4-column grid) with vintage frames
- Contact information with classic styling
- Footer with barber stripes
- Green WhatsApp floating button
- Classic booking modal

## Installation

```bash
npm install
npm run dev
```

Opens on port 3002 to avoid conflicts.

## Customization

All placeholder content is marked with `{{VARIABLE_NAME}}` for easy replacement via the deployment script.

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons
