# Barbershop Template - Modern Variant

Clean, minimalist design with a professional aesthetic.

## Design Features
- **Color Scheme**: White/light gray background with deep blue (#1E40AF) accents
- **Typography**: Inter & Poppins sans-serif fonts
- **Layout**: Single column with generous whitespace
- **Style**: Clean, contemporary, professional

## Sections
- Hero section with dual-column layout
- Services grid (2 columns)
- About with stats
- Gallery (8 images, 4-column grid)
- Contact information with map
- Footer with social links
- WhatsApp floating button
- Booking modal

## Installation

```bash
npm install
npm run dev
```

## Customization

All placeholder content is marked with `{{VARIABLE_NAME}}` for easy replacement via the deployment script.

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons
